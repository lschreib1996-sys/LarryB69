## Ⅰ. DIE NEUROWISSENSCHAFT — Warum Texte süchtig machen

### Ⅰ.A Das Broca-Areal — Sprachverarbeitung in Echtzeit

```
Gehörter Text
    ↓
Broca-Areal (Sprachverarbeitung) + Wernicke-Areal (Sinn-Verständnis)
    ↓
Präfrontaler Cortex (Vorhersage: welches Wort kommt als nächstes?)
    ↓
─── WENN VORHERSAGE ZUTRIFFT ───→ Dopamin (Belohnung)
─── WENN ÜBERRASCHUNG EINTRITT ─→ Noradrenalin (Aufmerksamkeit!)
```

**Schlüsselerkenntnis:** Dein Gehirn rät ständig das NÄCHSTE Wort. Trifft die Vorhersage zu → Belohnung. Weicht der Text clever ab → AUFMERKSAMKEIT. Die perfekte Zeile macht BEIDES: 70% vorhersehbar, 30% überraschend.

### Ⅰ.B Warum Reime süchtig machen (Wikipedia belegt)

Wikipedia definiert Reim als **"Wiederholung ähnlicher Klänge am Ende betonter Silben."** Das Gehirn erkennt das Muster nach 1–2 Zeilen und beginnt, den nächsten Reim VORHERZUSAGEN. Trifft er ein = Dopamin.

**Reim-Typen nach Wirkung:**

| Reim-Typ | Definition | Gehirn-Effekt |
|----------|-----------|---------------|
| **Vollreim** (Haus/Maus) | Konsonant+Vokal+Konsonant identisch | Starke Belohnung, 100% Vorhersage |
| **Halbreim** (Haus/raus) | Nur Vokal identisch | Leichte Überraschung → Aufmerksamkeit |
| **Augenreim** (Liebe/triebe) | Nur geschrieben ähnlich | Irritation beim HÖREN (funktioniert nicht gesungen) |
| **Identischer Reim** (Haus/Haus) | Gleiches Wort | LANGEWEILE (vermeiden) |
| **Kein Reim (Blankvers)** | Keine Reimstruktur | Unruhe, keine Belohnung (für Pop ungeeignet) |

### Ⅰ.C Die 4-Sekunden-Regel

Das menschliche Arbeitsgedächtnis (Broca-Areal) kann **4 Sekunden** gesprochene Sprache speichern. Texte, die länger als 4 Sekunden ohne erkennbare Struktur (Reim, Wiederholung, Rhythmus) sind, VERLIEREN den Hörer.

```
Pro Song-Zeile: max. 3–4 Sekunden Sprechdauer
Längere Zeilen → Gehirn "verliert" den Anfang
```

## Ⅱ. DIE FORMEL — Aufbau eines Sucht-Textes

### Ⅱ.A Die Hook-Text-Formel

```
Hook = (Zahl/Widerspruch) + (Körperteil/Sinn) + (universelles Gefühl)

BEISPIELE:
"I got 99 problems but a bitch ain't one"        ← Zahl + Widerspruch
"I can't feel my face when I'm with you"          ← Körperteil + Unmöglichkeit
"My heart will go on"                             ← Körperteil + Ewigkeit
"We will rock you"                                ← "Wir" + physische Aktion

FORMEL: [Konkretes Nomen] [Unmögliche/Bildhafte Aktion]
```

### Ⅱ.B Die Strophen-Formel

```
ZEILE 1: Situation beschreiben (WER, WO)
ZEILE 2: Konflikt einführen (ABER, DOCH, NICHT)
ZEILE 3: Emotion benennen (GEFÜHL, Farbe, Temperatur)
ZEILE 4: Hook andeuten (1 Wort der Hook fällt)

BEISPIEL (nach Formel):
"Ich steh' im Regen vor dei'm Haus"               ← Z1: Situation (WO)
"Aber du machst die Tür nicht auf"                ← Z2: Konflikt (ABER)
"Und mir wird kalt — nicht nur vom Wind"          ← Z3: Emotion (Kälte)
"Ich glaub', ich fühl'... mich..."                ← Z4: Hook-Andeutung

→ HOOK: "Ich fühl' mich leer ohne dich"
```

## Ⅲ. DIE 10 SÜCHTIG-MACHENDEN TEXT-TECHNIKEN

### Ⅲ.1 Anapher (Wiederholung am Zeilenanfang)

```
"Ich will atmen, ich will fliegen, ich will leben"

→ Gehirn erwartet nach 2× "Ich will" ein drittes "Ich will"
→ Trifft es ein = Dopamin
→ Merkt es sich = Ohrwurm-Potenzial
```

### Ⅲ.2 Epipher (Wiederholung am Zeilenende)

```
"Du sagst, du liebst mich — wirklich?
Du schwörst, du brauchst mich — wirklich?"

→ Das "wirklich?" wird zum Catch-Wort
→ Nach dem Song fragst DU dich "wirklich?"
```

### Ⅲ.3 Enjambement (Zeilensprung)

```
"Ich pack' meine Sachen und
geh' — nein, ich bleib'"

→ Zeilenumbruch an FALSCHER Stelle
→ Gehirn erwartet "und GEHE" — bekommt "und... GEH — NEIN ICH BLEIB"
→ DOPPELTE Dopamin-Ausschüttung (Überraschung + Auflösung)
```

### Ⅲ.4 Ellipse (Weglassen)

```
"Ich wollt' dir sagen... (Pause)... ach, vergiss es"

→ Hörer füllt die Lücke SELBST
→ Aktive Teilnahme = stärkere Bindung an den Text
→ Dein Gehirn denkt: "Das ist MEIN Gedanke!"
```

### Ⅲ.5 Antithese (Gegensatz-Paar)

```
"Du bist das Feuer — ich bin das Eis"
"Je näher ich dir komm', desto weiter gehst du weg"

→ Zwei unvereinbare Dinge in EINER Zeile
→ Gehirn muss arbeiten, um den Widerspruch zu lösen
→ Dopamin bei "Verstanden!"
```

### Ⅲ.6 Synästhesie (Sinne kreuzen)

```
"Ich hör' die Stille schreien"
"Deine Worte schmecken nach Betrug"
"Die Farbe deiner Stimme"

→ "Hören + Schreien" inkompatibel
→ Gehirn: "Das geht nicht — ABER ICH VERSTEHE WAS GEMEINT IST"
→ Mehrere Hirnareale gleichzeitig aktiv = stärkere Erinnerung
```

### Ⅲ.7 Alliteration (gleicher Anfangsbuchstabe)

```
"Fremde Freunde fallen flach"

→ Nicht inhaltlich — sondern KLANGLICH catchy
→ Alle "F"-Laute aktivieren dieselbe Neuronengruppe = "Feuer zusammen"
```

### Ⅲ.8 Assonanz (Vokal-Wiederholung)

```
"Der Mond steht hoch, der Tod kommt noch"

→ Lange O-Laute (Mond, hoch, Tod, noch)
→ Gleicher Vokal = gleiche Mundstellung = Gehirn erkennt das Muster UNBEWUSST
```

### Ⅲ.9 Onomatopöie (Lautmalerei)

```
"Boom, da war's passiert"
"Tick-tack, die Zeit verrinnt"

→ Wort-Klang entspricht Wort-Bedeutung
→ PRIMITIVE Gehirnregionen aktiviert (nicht Sprache — SOUND)
```

### Ⅲ.10 Die unerwartete Silbenzahl

```
ZEILE 1: "Und ich warte"              (4 Silben)
ZEILE 2: "Und ich warte hier"          (5 Silben → okay)
ZEILE 3: "Und ich warte hier auf dich" (7 Silben → Spannung steigt)
ZEILE 4: "...dich"                     (1 Silbe → Überraschung!)
```

→ Dein Gehirn erwartet 7 Silben in Zeile 4
→ Bekommt 1 Silbe
→ Muster-Bruch = AUFMERKSAMKEIT 💉

## Ⅴ. DAS REZEPT — Eine Strophe in 60 Sekunden

```
VORLAGE:

1. Nimm ein Motiv (z.B. Liebe + Verlust)
2. Schreibe EINEN Satz, der alles sagt (das wird der HOOK)
3. Strophe: beantworte WER, WO, WAS, WARUM in 4 Zeilen
4. JEDE Zeile endet mit einem Reimwort
5. Zeile 4 enthält 1 Wort der Hook

BEISPIEL:

Hook:               "Ohne dich ist alles nur halb"

Strophe:
1. Wer:             "Ich sitz' hier an unserm Tisch"
2. Wo/Wann:         "Da wo du immer neben mir saßt"
3. Was ist anders:  "Und alles schmeckt nach nichts — kein Witz"
4. Hook-Andeutung:  "Vielleicht weil es nur die Hälfte war"

→ REIM: Tisch/nichts, saßt/war/Hälfte... UNVOLLKOMMEN = Spannung
```

## Ⅵ. CHECKLISTE — Ist dein Text süchtig?

- [ ] Hook in den ersten 10 Sekunden hörbar
- [ ] Hook = max. 7 Wörter (Arbeitsgedächtnis-Grenze)
- [ ] **1 ungewöhnliches Wort** in der Hook (z.B. "Toxic", "Halo", "Firework")
- [ ] Verse: 4 Zeilen, Zeile 4 = Hook-Andeutung
- [ ] Reimschema: AABB oder ABAB (Pop-Standard)
- [ ] Jede 4. Zeile bricht das Schema LEICHT
- [ ] **1 Widerspruch** pro Strophe (Feuer/Eis, nah/fern)
- [ ] **1 Körperteil** im ganzen Song (Herz, Augen, Haut, Hände)
- [ ] **1 Zahl** oder **1 Farbe** im ganzen Song
- [ ] Brücke: komplett anderes Reimschema (Überraschung!)

## Ⅶ. WAS NICHT FUNKTIONIERT

- ❌ **"Herz/Schmerz"** — abgenutzt, Dopamin-frei (keine Überraschung)
- ❌ **Abstrakte Konzepte ohne Bild** — "Die Ewigkeit der Liebe in der Unendlichkeit" → Gehirn kann nichts "sehen"
- ❌ **Perfekter Paarreim 100% des Songs** — Langeweile nach Strophe 2
- ❌ **>7 Wörter pro Hook** — Gehirn speichert es nicht
- ❌ **kein einziges ungewöhnliches Wort** — zu glatt = keine Aufmerksamkeit
